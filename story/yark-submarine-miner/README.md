# Yark: Cerebrium Run

A Tauri action RPG where Yark pilots a submarine through Enceladus's living caverns to mine Cerebrium.

## Play

- Move with `WASD` or arrow keys.
- Hold `Space` near a glowing Cerebrium node to mine.
- Press `R` inside the Abyssus dock ring to bank cargo, refill oxygen, and repair.
- Spend credits on mining, engine, plating, and scanner modules.
- Avoid thermal vents and Deep sentries.

## Commands

```powershell
npm.cmd install
npm.cmd run dev
npm.cmd run tauri:build
```

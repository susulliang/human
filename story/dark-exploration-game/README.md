# Dark: Resonance Path

A minimalist puzzle-solving RPG built as a Tauri app. You play as Dark while she follows a future resonance toward Jark, solving quiet rooms about symbiosis, severance, analog memory, and the living vein.

The game takes inspiration from minimal mobile exploration games: compact screens, tactile choices, soft palettes, and puzzles that are solved by reading the room instead of fighting it.

## Commands

```powershell
npm.cmd install
npm.cmd run dev
npm.cmd run tauri:build
```

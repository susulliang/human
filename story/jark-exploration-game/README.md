# Jark: The Silent Vein

A simple web-based exploration game built as a Tauri app. You play as Jark, moving through Freetown, Enceladus, and Dark's Dream. Your choices shape the ending toward one of the three story variants in `../dark-and-jark`:

- `V1 - Canonical Descent`
- `V2 - Submerge`
- `V3 - Bearer`

## Commands

```powershell
npm.cmd install
npm.cmd run dev
npm.cmd run tauri:build
```

The Tauri project lives in this folder so the original universe bible remains intact.

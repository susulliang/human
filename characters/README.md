# Characters — Human-Ever-After

Canonical profiles for named figures in the universe bible. Each file is YAML with shared fields:

| Field | Description |
| :--- | :--- |
| `id` | Stable slug (filename without extension) |
| `name` | Display name |
| `born` / `died` | Year (CE); `null` if unknown or ongoing |
| `status` | e.g. `alive`, `deceased`, `vat-king`, `relocated` |
| `type` | `human`, `vat-king`, `flux-seer`, `ai` |
| `affiliations` | Factions, courts, cartels |
| `roles` | Titles and functions |
| `skills` | Capabilities and expertise |
| `personality` | Trait list |
| `biography` | Short in-universe summary |
| `era_active` | When they matter in the timeline |
| `canon` | Source markdown paths |
| `media` | In-universe films / stories (optional) |
| `legacy` / `arc` | Posthumous influence or narrative structure (optional) |
